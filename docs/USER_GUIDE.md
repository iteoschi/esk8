# User Guide

Power on
1) Power the board and remote.
2) Wait for the remote to show RF OK.

Controls
- Trigger = deadman switch. Must be held for acceleration. Braking still works without it.
- Short press trigger (neutral throttle) cycles pages.
- Long press trigger (neutral throttle) enters Settings mode.

Pages
- Ride: Speed, board battery %, remote battery %, RF status + lights/cruise flags.
- Stats: Trip distance, Wh/km, max speed.
- Energy: Input current, motor current, temps, range estimate, amp-hours.
- Config: Lights toggle and Cruise toggle.

Settings mode
- Long press to enter/exit.
- Move throttle up/down to navigate.
- Short press to edit a value (or run an action).
- In edit mode, move throttle up/down to change the value.

Safety
- If RF is lost, the board ramps down current to zero.
- Binding is only accepted when the board is not moving.
